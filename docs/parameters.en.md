# Parameters

!!! tip "Tips"

    For details of the style related parameters, please see: <https://manual.cytoscape.org/en/stable/Styles.html>.

## Reader

| Parameter | Type | Default | Description |
| ---------- | ---- | ------ | ----------- |
| xgmml_path | str  |        | XGMML path  |
